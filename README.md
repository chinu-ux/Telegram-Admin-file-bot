# Telegram File Store Bot (Admin Only + Temporary Access)

A Python Telegram bot that:
- Allows only the admin to upload media files
- Saves them in a private channel
- Generates permanent share links
- Users can access files for 1 hour (auto-deletes from chat)
- Tracks view stats per file
